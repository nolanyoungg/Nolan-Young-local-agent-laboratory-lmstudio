# GitHub Repository Review

id: github-repo-review
defaultSkills: evidence-based-review, repo-auditor
allowedTools: list_files, read_file, read_file_metadata, search_text
maxSteps: 36

Perform a read-only, evidence-based audit of the selected repository. Review structure, core logic, error handling, tests, configuration, scripts, documentation accuracy, and hygiene. Cite only inspected paths. Do not claim tests, commands, profiling, or network checks were run. Return limitations whenever source evidence is insufficient.
